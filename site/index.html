Ordered List

List of steps to follow to upload your code to github

  1. Open your browser and search github.com
  2. Create your github account
    1. create your own new repository
    2. copy the link of your repository
  3. Now open your command prompt
  4. Type git clone and paste that link and press enter
  5. Then make a new directory in your computer
  6. Make a new file in that directory
  7. Then open that file in your code editor and write your code
  8. Now save that file with .html extension
  9. Now go to that directory in which the file is situated
  10. Type git status
  11. Now type git add .
  12. Now type git commit -m "type your message"
  13. Now type git push

Congratulatons Now your code is uploaded to github
